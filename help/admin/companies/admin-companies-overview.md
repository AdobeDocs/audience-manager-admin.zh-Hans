# 公司 {#companies}

使用[!UICONTROL Companies]页面视图列表配置中的公司。 您可以编辑或删除现有公司或创建新公司，前提是您已分配了相应的用户角色。

![](assets/companies.png)

单击所需列的标题，可以按升序或降序对每列进行排序。

使用[!UICONTROL Search]框或列表底部的分页控件找到所需的公司。

# 公司 {#companies}

使用 [!UICONTROL Companies] 页面查看Audience Manager配置中的公司列表。您可以根据分配的用户角色编辑或删除现有公司或创建新公司。

![](assets/companies.png)

您可以单击所需列的标题，以升序或降序排列每个列。

使用 [!UICONTROL Search] 列表底部的分页控件或分页控件查找所需的公司。

# 公司 {#companies}

使用 [!UICONTROL Companies] 页面查看Audience Manager配置中的公司列表。 您可以编辑或删除现有公司或创建新公司，前提是您分配了相应的用户角色。

![](assets/companies.png)

您可以通过单击所需列的标题，按升序或降序对每个列进行排序。

使用 [!UICONTROL Search] 框或列表底部的分页控件以查找所需的公司。

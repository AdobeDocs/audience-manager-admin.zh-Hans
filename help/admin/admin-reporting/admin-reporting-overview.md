# 报表 {#reporting}

通过创建新报表或编辑或删除现有报表来管理Audience Manager报告。您还可以以 [!DNL Tableau] 用户身份分配门户用户。

<!-- c_reporting.xml -->

![](assets/reporting.png)

您可以单击所需列的标题，以升序或降序排列每个列。

使用 [!UICONTROL Search] 该框或列表底部的分页控件查找所需的报告。

# 报表 {#reporting}

通过创建新报告或编辑或删除现有报告来管理Audience Manager报告。 还可以将门户用户分配为[!DNL Tableau]用户。

<!-- c_reporting.xml -->

![](assets/reporting.png)

单击所需列的标题，可以按升序或降序对每列进行排序。

使用[!UICONTROL Search]框或列表底部的分页控件找到所需的报告。
